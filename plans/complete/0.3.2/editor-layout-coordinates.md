# Editor 布局位置关系

**版本**: 0.3.2

---

## Screen 级布局

假定屏幕尺寸 = W × H

```
        0,0                                           W,0
         ┌──────────────────────────────────────────────┐
         │                                               │
    24px │              MenuBar (0, 0, W, 24)            │
         │                                               │
    menuH├──────────┬────────────────────────────────────┤
         │          │                                    │
         │          │                                    │
         │ LeftPanel│           Preview                  │
         │          │                                    │
  previewH│ (0,menuH,│   (leftW, menuH, W-leftW, previewH)│
         │  leftW,  │                                    │
         │ previewH)│                                    │
         │          │                                    │
  menuH+ ├──────────┴────────────────────────────────────┤
 previewH│                                               │
         │            Timeline (0, y, W, timelineH)       │
         │                                               │
         └──────────────────────────────────────────────┘
        0,H                                            W,H
```

```
menuH     = 24
leftW     = W * 0.2
timelineH = H * 0.22
previewH  = H - menuH - timelineH
timelineY = menuH + previewH
```


## LeftPanel

位置: `(0, menuH)` — `(leftW, menuH + previewH)`

内部动态生成，内容高度超出时右侧 4px 滚动条。

---

## Preview

位置: `(leftW, menuH)` — `(W, menuH + previewH)`

```
┌───────────────────────────────────────┐
│                                       │
│          FBO 纹理 (16:9 内接)          │
│                                       │
│        ▶    ⏸    ⏹    0.0s            │  ← 底部居中，距底边8px
│       barX  +40  +80    +124          │     barW=160, barH=24
└───────────────────────────────────────┘
```

```
barX = leftW + (previewW - 160) / 2
barY = menuH + previewH - 24 - 8
```

▶ 按钮: `(barX, barY, 36, 24)`
⏸ 按钮: `(barX+40, barY, 36, 24)`
⏹ 按钮: `(barX+80, barY, 36, 24)`
时间标签: `(barX+124, barY+7)`

---

## Timeline

位置: `(0, menuH + previewH)` — `(W, H)`

```
┌───┬──────┬───────────────────────────────────────────┐
│   │      │  刻度尺 (高 20)                             │
│工  │ 轨道  ├───────────────────────────────────────────┤
│具  │ 标签  │  Camera  ▓▓▓▓▓▓▓▓▓▓▓▓▓▓▓▓▓▓▓▓▓▓▓▓▓▓▓  │ 轨道高 28
│栏  │      │  LTRBOX  ▓▓▓▓▓▓▓▓▓▓▓▓▓▓▓▓▓▓▓▓▓▓▓▓▓▓▓  │
│22  │ 58   │                                           │
└───┴──────┴───────────────────────────────────────────┘
```

```
TOOLBAR_W = 22     工具栏按钮列宽
LABEL_W   = 58     轨道标签列宽
LEFT_W    = 80     左侧总宽度 (22+58)
HEADER_H  = 20     刻度尺行高
TRACK_H   = 28     每条轨道行高
RESIZE_MARGIN = 4  clip 左右拖拽手柄宽
```

**画布区域**: 起点 `(areaX + 80, areaY + 20)`，宽 `W - 80`

**工具栏按钮** (bx = areaX + 3，从上到下排列):
```
+C (16x16)
-C (16x16)    间距 2px, 分组间距 4px
+K (16x16)
-K (16x16)
«» (16x16)    吸附
```

**刻度尺**: time→pixel = `canvasX + time*60 + scrollOffset`
**放大**: Ctrl+滚轮, `pixelsPerSecond` 范围 [10, 500]
