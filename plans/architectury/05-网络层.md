# 阶段 5：网络层

## 目标

迁移网络层（trigger/network/）到 common 模块，将 Forge 的 `SimpleChannel` 替换为 Architectury 的 `NetworkManager`。

## 涉及文件

### 旧文件 (old/src/main/java/)

**trigger/network/**
| 文件 | Forge 依赖 | 迁移说明 |
|------|-----------|---------|
| `NetworkHandler.java` | `SimpleChannel`, `NetworkRegistry`, `NetworkEvent`, `NetworkDirection`, `DistExecutor` | **完全重写** |
| `S2CPlayScriptPacket.java` | 无 | 纯 Vanilla，保留 |
| `S2CStopScriptPacket.java` | 无 | 纯 Vanilla，保留 |
| `S2CTriggerStateSyncPacket.java` | 无 | 纯 Vanilla，保留 |
| `S2CSkipVoteUpdatePacket.java` | 无 | 纯 Vanilla，保留 |
| `C2SScriptFinishedPacket.java` | `NetworkEvent.Context` | 需改造 |
| `C2SPlaybackStartedPacket.java` | `NetworkEvent.Context` | 需改造 |

## 旧架构网络分析

旧系统使用 Forge `SimpleChannel` 注册了以下 6 个包：

| 方向 | 包 | 用途 |
|------|---|------|
| S2C | `S2CPlayScriptPacket` | 服务器通知客户端播放脚本 |
| S2C | `S2CStopScriptPacket` | 服务器通知客户端停止脚本 |
| S2C | `S2CTriggerStateSyncPacket` | 同步触发器状态到客户端 |
| S2C | `S2CSkipVoteUpdatePacket` | 更新跳过投票计数 |
| C2S | `C2SScriptFinishedPacket` | 客户端通知服务器脚本播放完毕 |
| C2S | `C2SPlaybackStartedPacket` | 客户端通知服务器脚本开始播放 |

## 迁移步骤

### Step 1: 转换 Packet 类为 CustomPacketPayload

Architectury 使用 Vanilla 的 `CustomPacketPayload` 接口替代 Forge 的序列化方式。

**改造后网络包结构**（以 `S2CPlayScriptPacket` 为例）：

```java
// common/ - S2CPlayScriptPacket.java
public record S2CPlayScriptPacket(String scriptId) implements CustomPacketPayload {
    public static final ResourceLocation TYPE_ID = 
        new ResourceLocation(ImmersiveCinematics.MOD_ID, "play_script");
    public static final StreamCodec<FriendlyByteBuf, S2CPlayScriptPacket> CODEC = 
        StreamCodec.ofMember(S2CPlayScriptPacket::write, S2CPlayScriptPacket::new);

    public S2CPlayScriptPacket(FriendlyByteBuf buf) {
        this(buf.readUtf());
    }

    public void write(FriendlyByteBuf buf) {
        buf.writeUtf(scriptId);
    }

    @Override
    public ResourceLocation type() {
        return TYPE_ID;
    }
}
```

**所有 6 个包的改造要点**：
- 实现 `CustomPacketPayload` 接口
- 定义 `TYPE_ID`（ResourceLocation）
- 定义 `CODEC`（StreamCodec）
- 实现 `type()` 方法
- 删除对 `NetworkEvent.Context` 的引用
- 将 `handle()` 方法改为接收 `PacketContext` 参数

### Step 2: 重写 NetworkHandler

```java
// common/ - NetworkHandler.java
public class NetworkHandler {
    public static void register() {
        // 注册 C2S 接收器（服务器端）
        NetworkManager.registerReceiver(NetworkManager.Side.C2S,
            C2SScriptFinishedPacket.TYPE, C2SScriptFinishedPacket.CODEC,
            (packet, context) -> context.queue(() -> {
                ServerPlayer player = (ServerPlayer) context.getPlayer();
                C2SScriptFinishedPacket.handle(packet, player);
            }));

        NetworkManager.registerReceiver(NetworkManager.Side.C2S,
            C2SPlaybackStartedPacket.TYPE, C2SPlaybackStartedPacket.CODEC,
            (packet, context) -> context.queue(() -> {
                ServerPlayer player = (ServerPlayer) context.getPlayer();
                C2SPlaybackStartedPacket.handle(packet, player);
            }));

        // 声明 S2C 载荷类型（用于发送）
        NetworkManager.registerS2CPayloadType(S2CPlayScriptPacket.TYPE, S2CPlayScriptPacket.CODEC);
        NetworkManager.registerS2CPayloadType(S2CStopScriptPacket.TYPE, S2CStopScriptPacket.CODEC);
        NetworkManager.registerS2CPayloadType(S2CTriggerStateSyncPacket.TYPE, S2CTriggerStateSyncPacket.CODEC);
        NetworkManager.registerS2CPayloadType(S2CSkipVoteUpdatePacket.TYPE, S2CSkipVoteUpdatePacket.CODEC);
    }

    // S2C 发送方法
    public static void sendToPlayer(ServerPlayer player, Object packet) {
        if (packet instanceof CustomPacketPayload payload) {
            NetworkManager.sendToPlayer(player, payload);
        }
    }

    // C2S 发送方法（客户端调用）
    public static void sendToServer(Object packet) {
        if (packet instanceof CustomPacketPayload payload) {
            NetworkManager.sendToServer(payload);
        }
    }
}
```

### Step 3: 注册 S2C 客户端接收器

S2C 包的接收在客户端注册：

```java
// 在客户端初始化中（fabric 和 forge 共用）
public static void registerClientReceivers() {
    NetworkManager.registerReceiver(NetworkManager.Side.S2C,
        S2CPlayScriptPacket.TYPE, S2CPlayScriptPacket.CODEC,
        (packet, context) -> context.queue(() -> {
            ClientScriptReceiver.handlePlayScript(packet);
        }));

    NetworkManager.registerReceiver(NetworkManager.Side.S2C,
        S2CStopScriptPacket.TYPE, S2CStopScriptPacket.CODEC,
        (packet, context) -> context.queue(() -> {
            ClientScriptReceiver.handleStopScript(packet);
        }));

    NetworkManager.registerReceiver(NetworkManager.Side.S2C,
        S2CTriggerStateSyncPacket.TYPE, S2CTriggerStateSyncPacket.CODEC,
        (packet, context) -> context.queue(() -> {
            ClientTriggerStateCache.handleSync(packet);
        }));

    NetworkManager.registerReceiver(NetworkManager.Side.S2C,
        S2CSkipVoteUpdatePacket.TYPE, S2CSkipVoteUpdatePacket.CODEC,
        (packet, context) -> context.queue(() -> {
            ClientScriptReceiver.handleSkipVoteUpdate(packet);
        }));
}
```

### Step 4: C2S 包改造

C2S 包中原来使用 `NetworkEvent.Context` 获取玩家和设置包已处理标志。

Architectury 中，`PacketContext` 通过 `context.getPlayer()` 获取玩家，`context.queue()` 控制主线程执行。

```java
// C2SScriptFinishedPacket.java - handle 改造
public record C2SScriptFinishedPacket(String scriptId, UUID playerUuid) 
        implements CustomPacketPayload {
    // ... 定义 CODE, CODEC ...

    public static void handle(C2SScriptFinishedPacket packet, ServerPlayer player) {
        ScriptEventManager.INSTANCE.onScriptFinished(player, packet.scriptId);
    }
}
```

### 客户端注册位置

由于 S2C 接收器只能在客户端注册，需要：
- `fabric/` 中在 `ImmersiveCinematicsFabricClient.onInitializeClient()` 调用 `registerClientReceivers()`
- `forge/` 中在客户端初始化事件中调用 `registerClientReceivers()`

```java
// fabric/
public class ImmersiveCinematicsFabricClient implements ClientModInitializer {
    @Override
    public void onInitializeClient() {
        NetworkHandler.registerClientReceivers();
    }
}

// forge/
// 在 Common init 中只注册 C2S，客户端接收器在 client setup 时注册
ClientLifecycleEvent.CLIENT_SETUP.register(mc -> {
    NetworkHandler.registerClientReceivers();
});
```
