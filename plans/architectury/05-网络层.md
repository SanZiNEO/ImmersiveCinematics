# 阶段 5：网络层

## 目标

迁移网络层（trigger/network/）到 common 模块，将 Forge 的 `SimpleChannel` 替换为 Architectury 的 `SimpleNetworkManager`（参考 FTB-Quests 模式）。

## 旧架构网络分析

旧系统使用 Forge `SimpleChannel` 注册了以下 6 个包：

| 方向 | 包 | 用途 |
|------|---|------|
| S2C | `S2CPlayScriptPacket` | 服务器通知客户端播放脚本 |
| S2C | `S2CStopScriptPacket` | 服务器通知客户端停止脚本 |
| S2C | `S2CTriggerStateSyncPacket` | 同步触发器状态到客户端 |
| S2C | `S2CSkipVoteUpdatePacket` | 更新跳过投票计数 |
| C2S | `C2SScriptFinishedPacket` | 客户端通知服务器脚本播放完毕 |
| C2S | `C2SPlaybackStartedPacket` | 客户端通知服务器脚本开始播放 |

## 迁移步骤

### Step 1: 声明 SimpleNetworkManager

```java
// common/ - NetworkHandler.java (参考 FTBQuestsNetHandler)
public interface NetworkHandler {
    SimpleNetworkManager NET = SimpleNetworkManager.create(ImmersiveCinematics.MOD_ID);

    // 注册所有包类型
    MessageType PLAY_SCRIPT = NET.registerS2C("play_script", S2CPlayScriptPacket::new);
    MessageType STOP_SCRIPT = NET.registerS2C("stop_script", S2CStopScriptPacket::new);
    MessageType TRIGGER_STATE_SYNC = NET.registerS2C("trigger_state_sync", S2CTriggerStateSyncPacket::new);
    MessageType SKIP_VOTE_UPDATE = NET.registerS2C("skip_vote_update", S2CSkipVoteUpdatePacket::new);
    MessageType SCRIPT_FINISHED = NET.registerC2S("script_finished", C2SScriptFinishedPacket::new);
    MessageType PLAYBACK_STARTED = NET.registerC2S("playback_started", C2SPlaybackStartedPacket::new);

    static void init() {
        // SimpleNetworkManager 会在两个平台自动注册
        // 只需声明 MessageType 即可——初始化时 static 字段会加载
    }
}
```

### Step 2: 改造 Packet 类为 BaseS2CMessage / BaseC2SMessage

以 S2C 为例，继承 `BaseS2CMessage`：

```java
// common/ - S2CPlayScriptPacket.java
public class S2CPlayScriptPacket extends BaseS2CMessage {
    private final String scriptId;

    // 解码构造函数（SimpleNetworkManager 自动调用）
    public S2CPlayScriptPacket(FriendlyByteBuf buf) {
        this.scriptId = buf.readUtf();
    }

    // 编码构造函数
    public S2CPlayScriptPacket(String scriptId) {
        this.scriptId = scriptId;
    }

    @Override
    public MessageType getType() {
        return NetworkHandler.PLAY_SCRIPT;
    }

    @Override
    public void write(FriendlyByteBuf buf) {
        buf.writeUtf(scriptId);
    }

    @Override
    public void handle(NetworkManager.PacketContext context) {
        // 客户端侧处理（不需要 context.queue()，BaseS2CMessage 自动在主线程执行）
        ClientScriptReceiver.handlePlayScript(this);
    }
}
```

C2S 消息继承 `BaseC2SMessage`：

```java
// common/ - C2SScriptFinishedPacket.java
public class C2SScriptFinishedPacket extends BaseC2SMessage {
    private final String scriptId;

    public C2SScriptFinishedPacket(FriendlyByteBuf buf) {
        this.scriptId = buf.readUtf();
    }

    public C2SScriptFinishedPacket(String scriptId) {
        this.scriptId = scriptId;
    }

    @Override
    public MessageType getType() {
        return NetworkHandler.SCRIPT_FINISHED;
    }

    @Override
    public void write(FriendlyByteBuf buf) {
        buf.writeUtf(scriptId);
    }

    @Override
    public void handle(NetworkManager.PacketContext context) {
        // 服务端侧处理
        ServerPlayer player = (ServerPlayer) context.getPlayer();
        ScriptEventManager.INSTANCE.onScriptFinished(player, scriptId);
    }
}
```

**所有 6 个包的改造要点**：
- 删除 Forge 的 `NetworkEvent.Context`、`FriendlyByteBuf` 手动序列化等
- 继承 `BaseS2CMessage` 或 `BaseC2SMessage`（根据方向）
- 实现 `getType()` → 返回 `NetworkHandler` 中声明的 `MessageType`
- 实现 `write(FriendlyByteBuf)` → 编码
- 构造方法 `(FriendlyByteBuf)` → 解码
- `handle(NetworkManager.PacketContext)` → 业务处理

### Step 3: 发送调用

SimpleNetworkManager 的消息发送通过消息实例上的方法：

```java
// 服务端发送到玩家
new S2CPlayScriptPacket("script_id").sendTo(player);

// 服务端发送到所有玩家
new S2CPlayScriptPacket("script_id").sendToAll(server);

// 客户端发送到服务器
new C2SScriptFinishedPacket("script_id").sendToServer();
```

对比 Forge 旧写法：
```java
// Forge 旧写法
NetworkHandler.CHANNEL.sendTo(packet, player.connection.connection, NetworkDirection.PLAY_TO_CLIENT);
NetworkHandler.CHANNEL.sendToServer(packet);
```

### Step 4: @ExpectPlatform 处理平台差异（如果需要）

如果某些序列化操作有平台差异（如物品栈的 ForgeCaps），用 `@ExpectPlatform`：

```java
// common/ - NetworkHandler.java
@ExpectPlatform
static void writeItemStack(FriendlyByteBuf buf, ItemStack stack) {
    throw new AssertionError();
}

@ExpectPlatform
static ItemStack readItemStack(FriendlyByteBuf buf) {
    throw new AssertionError();
}
```

```java
// fabric/.../net/NetworkHandlerImpl.java
public static void writeItemStack(FriendlyByteBuf buf, ItemStack stack) {
    buf.writeItem(stack);
}
public static ItemStack readItemStack(FriendlyByteBuf buf) {
    return buf.readItem();
}
```

```java
// forge/.../net/NetworkHandlerImpl.java
public static void writeItemStack(FriendlyByteBuf buf, ItemStack stack) {
    // Forge 需要处理 ForgeCaps
    if (stack.isEmpty()) { buf.writeVarInt(-1); return; }
    buf.writeVarInt(Item.getId(stack.getItem()));
    buf.writeNbt(stack.getTag());
    buf.writeNbt(stack.save(new CompoundTag()).getCompound("ForgeCaps"));
}
public static ItemStack readItemStack(FriendlyByteBuf buf) {
    int id = buf.readVarInt();
    if (id == -1) return ItemStack.EMPTY;
    CompoundTag tag = buf.readNbt();
    CompoundTag caps = buf.readNbt();
    ItemStack item = new ItemStack(Item.byId(id), 1, caps);
    item.setTag(tag);
    return item;
}
```

> **注意**：我们现有的 6 个包只传输 `String` 和 `int`，没有 `ItemStack`，所以目前不需要 `@ExpectPlatform`。这里作为参考模式记录。

### 注册说明

使用 `SimpleNetworkManager` 后，**不需要**手动注册 C2S/S2C 接收器。`SimpleNetworkManager` 自动在初始化时注册所有接收器。消息类的 `handle()` 方法会自动在两个平台上被调用。

需要确保：
- 在 `ImmersiveCinematics.init()` 中调用 `NetworkHandler.init()`（触发 static 字段加载）
- Fabric 端不需要额外操作（`SimpleNetworkManager` 自动初始化）
- Forge 端 `EventBuses.registerModEventBus()` 已调用即可
