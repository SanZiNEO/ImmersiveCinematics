### Step 2: RawInputLogger 改造

**当前状态**：使用 Forge `@EventBusSubscriber` + `InputEvent.Key` / `InputEvent.MouseButton` / `InputEvent.MouseScrollingEvent`。

**改造方案**：使用 Architectury `ClientRawInputEvent` 监听原生输入事件。

```java
// common/ - RawInputLogger.java
public class RawInputLogger {
    // 全部业务逻辑不变
    // 移除 @SubscribeEvent 注解
    // 将 @SubscribeEvent 方法变为静态方法

    public static void onKeyPress(int key, int scanCode, int action, int modifiers) {
        // ...
    }

    public static void onMouseButton(int button, int action, int modifiers) {
        // ...
    }

    public static void onMouseScroll(double delta) {
        // ...
    }

    public static void onClientTick() {
        // ...
    }
}
```

**注册方式**（在客户端事件注册中）：

```java
// 通过 ClientRawInputEvent 注册
ClientRawInputEvent.KEY_PRESSED.register((key, scanCode, action, modifiers) -> {
    if (enabled) RawInputLogger.onKeyPress(key, scanCode, action, modifiers);
    return EventResult.pass();
});

ClientRawInputEvent.MOUSE_CLICKED.register((button, action, modifiers) -> {
    if (enabled) RawInputLogger.onMouseButton(button, action, modifiers);
    return EventResult.pass();
});
```

> **注意**：`ClientRawInputEvent` 的具体 API 签名需查阅 Architectury 文档确认。如果 Architectury 没有提供完全等价的原始输入事件，考虑使用 Mixin 替代。

### Step 4: 最终主类整合

在此阶段，`common/ImmersiveCinematics.java` 的 `init()` 方法应包含所有模块的初始化：

```java
public static void init() {
    // === 基础 ===

    // === 网络 ===
    NetworkHandler.register();

    // === 触发器 ===
    TriggerRegistration.registerAll();
```

### Step 5: 跨平台入口点确认

**Fabric 入口**（`fabric/ImmersiveCinematicsFabric.java`）：
```java
public class ImmersiveCinematicsFabric implements ModInitializer {
    @Override
    public void onInitialize() {
        ImmersiveCinematics.init();
    }
}
```

**Fabric 客户端入口**（`fabric/client/ImmersiveCinematicsFabricClient.java`）：
```java
public class ImmersiveCinematicsFabricClient implements ClientModInitializer {
    @Override
    public void onInitializeClient() {
        registerKeyMappings();
        NetworkHandler.registerClientReceivers();
        ClientEventHandler.register();
        ClientGuiEvent.RENDER_HUD.register(SkipHudRenderer::render);
        ClientGuiEvent.RENDER_HUD.register((graphics, delta) -> 
            CinematicOverlay.render(graphics, ...));
    }
}
```

**Forge 入口**（`forge/ImmersiveCinematicsForge.java`）：
```java
@Mod(ImmersiveCinematics.MOD_ID)
public class ImmersiveCinematicsForge {
    public ImmersiveCinematicsForge() {
        EventBuses.registerModEventBus(ImmersiveCinematics.MOD_ID, 
            FMLJavaModLoadingContext.get().getModEventBus());
        ImmersiveCinematics.init();

        // Forge 特定的客户端初始化
        ModLoadingContext.get().registerExtensionPoint(
            ConfigScreenHandler.ConfigScreenFactory.class,
            () -> new ConfigScreenHandler.ConfigScreenFactory(
                (mc, parent) -> new ConfigScreen(parent)));
    }
}
```
