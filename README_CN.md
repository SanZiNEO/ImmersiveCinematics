# ImmersiveCinematics

**专业级 Minecraft 电影摄影相机引擎。** 通过 JSON 脚本和游戏内编辑器，创建震撼的过场动画、流畅的镜头运动和沉浸式叙事体验。

## 这是什么？

ImmersiveCinematics 是一个 Minecraft 模组（含客户端与服务端组件），让你完全掌控游戏相机。编写 JSON 脚本或使用游戏内编辑器，输入一条命令，就能让 Minecraft 变身电影级体验。

服务端组件提供触发器系统（位置、进度、击杀、交互等事件），可在指定条件满足时自动播放脚本。

**适用场景：**
- 带有流畅镜头运动的过场动画
- 区域固定视角（生化危机0式）
- 自动化世界巡游和飞行展示
- 游戏电影和影视制作
- 脚本化相机角度增强沉浸感

## 功能特性

**电影级相机**
- 6 自由度控制：位置、偏航、俯仰、滚转、视场角、缩放
- 景深（DOF）不在计划内——实现景深需要复杂的 OpenGL 管线集成，与模组的光影兼容设计理念不符
- 关键帧动画驱动的流畅镜头路径
- 贝塞尔曲线路径，创建优雅弧线运动
- 5 种插值曲线：线性、平滑、缓入、缓出、缓入缓出
- 镜头间交叉淡化过渡，或硬切换制造戏剧效果
- 可配置循环次数，或无限时长循环

**JSON 脚本系统**
- 多轨道时间轴：相机、黑边、音频、事件和模组事件轨道
- 相对或绝对相机定位
- 脚本抢占控制，适用于区域固定视角
- 严格验证，清晰的错误提示

**电影级呈现**
- 宽银幕黑边（默认 2.35:1），带渐入/渐出动画
- 自动隐藏 HUD、手臂和手持物品
- 视角摇晃抑制（受伤抖动、行走摇晃、反胃效果）
- 光影包兼容

**完全控制**
- 15 项运行时行为标志，精细调控
- 可跳过或强制观看的过场动画
- 末帧保持，适用于固定视角区域
- 暂停感知：游戏暂停时脚本自动冻结

## 快速开始

1. 安装 [Minecraft Forge 1.20.1](https://files.minecraftforge.net/net/minecraftforge/forge/index_1.20.1.html)（47.x+）
2. 下载 ImmersiveCinematics 并放入 `.minecraft/mods/`
3. 启动游戏

### 命令

| 命令 | 说明 |
|------|------|
| `/cinematic play <文件>` | 播放电影脚本 |
| `/cinematic stop` | 停止当前播放 |
| `/cinematic status` | 显示播放状态 |

### 游戏内编辑器

使用内置的时间轴编辑器，无需离开游戏即可可视化创建和修改脚本。（0.3.0 版本推出）
